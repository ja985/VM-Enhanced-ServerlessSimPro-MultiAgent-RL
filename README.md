
# VM-Aware Serverless Simulation Framework with MAPPO

This repository extends the original ServlessSimPro framework with:

- VM-aware serverless architecture
- PM -> VM -> Container -> Function hierarchy
- MAPPO-based intelligent scheduling
- Energy-aware resource consolidation
- Reinforcement learning-driven autoscaling

## Added Components

- virtual_machine.py
- mappo_scheduler.py
- VM-aware physical machine layer
- Extended energy-aware scheduling support

## Repository Structure

- resource/
- schedulingInterface/
- results/
- traces/
- metrics/

## Research Contribution

This framework supports scalable and energy-efficient serverless scheduling research using VM-aware resource abstraction and MAPPO-based scheduling.
